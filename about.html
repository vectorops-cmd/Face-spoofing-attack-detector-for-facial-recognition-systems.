LiveGuard AI

Advanced CNN-based face liveness detection system trained on multiple attack datasets.

System Overview

LiveGuard AI is a state-of-the-art face anti-spoofing system designed to detect and prevent presentation attacks including printed photos, replay videos, and 3D masks. The system uses a CNN model (ResNet-50 recommended) trained across CASIA FASD, ReplayAttack and 3D-mask datasets to improve cross-dataset generalization.

Training Datasets

  • CASIA FASD — print & replay
  • ReplayAttack — replay video attacks
  • 3D Mask Dataset(s) — physical mask attacks

Key Features

  • Multi-attack detection (print, replay, 3D mask)
  • Cross-dataset generalization
  • Realtime processing (local inference)
  • Logging, analytics & export

Technical Specs (example)

Model: ResNet-50
Input: 224x224 RGB
Output: live / spoof (plus attack type)
Performance target: high accuracy on combined datasets
